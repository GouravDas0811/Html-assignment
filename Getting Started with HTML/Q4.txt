HTML Tags: HTML tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever is written inside < and > are called tags.
 For example, in <b>, <b> is the starting tag and </b> is the ending tag1.
HTML Elements: The HTML element consists of both the opening and closing tags as well as what’s inside those tags1. It normally consists of some structure that’s used to define the respective tags. For instance, <b>This is the content.</b> is an HTML element where <b> is the starting tag, </b> is the ending tag, and This is the content. is the content inside the tags
